# Dokumentace — lidská vrstva

Tahle část repa je **pro lidi**, bez ohledu na AI. Najdeš tu, jak ve firmě
co děláme, jak se naloďuje nový člověk a návody k nástrojům, které používáme.

Cílem je, aby se každý dokázal rychle zorientovat a kvalita nestála na jednom
člověku.

## Co kde hledat

- **[`procesy/`](procesy/)** — jak co děláme. Workflow, postupy, osvědčené
  způsoby práce napříč firmou.
- **[`onboarding/`](onboarding/)** — jak se naloďuje nový člověk. První kroky,
  přístupy, co si nastavit a přečíst.
- **[`nastroje/`](nastroje/)** — návody k nástrojům, které používáme (Freelo,
  Claude Code a další).

Nové dokumenty pojmenovávej podle konvence v [CONTRIBUTING.md](../CONTRIBUTING.md)
(malá písmena, pomlčky, bez diakritiky) a zmiň je v `README.md` dané podsložky.

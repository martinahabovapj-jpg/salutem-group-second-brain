# Onboarding

Jak se u nás naloďuje nový člověk — první kroky, přístupy a co si nastavit
a přečíst, aby se rychle zorientoval.

## Co sem časem patří

- uvítací průvodce pro nového kolegu (kudy začít)
- seznam přístupů a nástrojů, které si má nastavit
- odkazy na klíčové procesy a kontext, které si má projít

Každý dokument dej do vlastního souboru pojmenovaného podle konvence
(malá písmena, pomlčky, bez diakritiky) — např. `prvni-den.md`.

# Procesy

Jak ve firmě co děláme — workflow, postupy a osvědčené způsoby práce.

Protože nemáme dedikovaného projektového manažera, zapisujeme si tady, jak
se věci dělají správně, ať kvalita nestojí na jednom člověku a každý ví, jak
na to.

## Co sem časem patří

- postupy pro opakované činnosti (např. jak založit projekt, jak předat práci)
- workflow napříč týmem
- pravidla a standardy, které chceme držet jednotně

Každý proces dej do vlastního souboru pojmenovaného podle konvence
(malá písmena, pomlčky, bez diakritiky) — např. `zalozeni-projektu.md`.

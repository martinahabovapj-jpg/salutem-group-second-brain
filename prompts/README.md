# Prompty a šablony

Tahle složka je na **znovupoužitelné prompty a šablony** pro Claude — texty,
které opakovaně kopíruješ do chatu, ať nezačínáš pokaždé od nuly.

Na rozdíl od skillů (které Claude Code spouští sám automaticky) jsou prompty
texty, které vědomě vezmeš a použiješ, když je potřebuješ.

## Co sem časem patří

- šablony zadání pro opakované úkoly (např. „shrň mi tenhle dokument takto…")
- osvědčené formulace, které dávají dobré výsledky
- strukturované prompty pro konkrétní situace

## Jak prompty pojmenovávat

- malá písmena, slova oddělená pomlčkami, bez diakritiky a mezer
- výstižně podle účelu — `shrnuti-dokumentu.md`, `email-klientovi.md`
- nahoru do souboru napiš krátce, k čemu prompt je a jak ho použít

# Firemní kontext pro Claude

Tahle složka drží **firemní kontext**, který přikládáme Claude, aby chápal,
pro koho a jak pracuje — kdo jsme, jak mluvíme, co která slova u nás znamenají.
Díky tomu nemusíme tyhle věci vysvětlovat pořád dokola.

## Co sem časem patří

- **`o-firme.md`** — kdo jsme, čím se zabýváme, jak fungujeme. Základní kontext,
  ze kterého Claude čerpá.
- **`slovnik-pojmu.md`** — jednotné názvosloví. Co u nás znamenají interní
  pojmy, zkratky, názvy projektů a nástrojů.
- **`styl-komunikace.md`** — jak píšeme a mluvíme. Tón, oslovení, čeho se
  držet a čemu se vyhnout (spisovná, lidská čeština, k věci, bez zbytečných
  odborností).

Soubory přidávej postupně podle konvence pojmenování (malá písmena, pomlčky,
bez diakritiky).
